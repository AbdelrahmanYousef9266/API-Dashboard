# API Dashboard

A modern JavaScript dashboard that integrates multiple public APIs to display real-time data in one place.

---

##  Features

-  Random Dog Images  
-  Random Cat Images  
-  Weather by City (temperature + wind speed)  
-  Currency Exchange (USD to EUR & JOD)  
-  GitHub User Search  
-  Random Jokes  
-  Trending Movies  
-  Random Advice  

---

##  Technologies Used

- HTML5  
- CSS3 (Grid + Flexbox)  
- JavaScript (ES6+)  
- Fetch API  
- Async / Await  

---

##  How It Works

Each section (card) connects to a different public API.  

When a user interacts with the app:

1. A request is sent using `fetch()`
2. The response is converted to JSON
3. Data is displayed dynamically on the page
